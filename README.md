### Подсчет голосов при голосовании в многоквартирном доме

Несколько раз наблюдал как жена считает голоса при голосовании в многоквартирном доме: долго, заползают ошибки, постоянный пересчет.

в приложении нет тестов, нормальной локализации и прочего всего что должно быть. задача была как на хакатон, и она была решена:
за рабочий день было создано приложение которое значительно упрощает жизнь.

* Ruby 3.4.5

```bash
# инициализация

bin/rails db:migrate
bin/rails db:seed
```


```bash
# для запуска два разных окна

bin/rails s
yarn build:css --watch
```
