### Подсчет голосов при голосовании в многоквартирном доме

Постановка задачи: реализовать подсчет голосов в зависимости от владеемой площади.

Суть проблемы: в некоторых квартирах несколько владельцев и голосование "по-головам" ущемляет права других владельцев. Подсчет в зависимости от владеемой площади вызывал множество трудностей.

Несколько раз наблюдал как жена считает голоса: долго, с ошибками, постоянный пересчет.

В приложении не до конца реализована локализация и тестирование, но задача была как на хакатон, и она была решена:
за рабочий день (около 6.5 часов + второй день около 2 часов на локализацию, linter, тесты) было создано полностью рабочее приложение которое значительно упрощает жизнь.

* Ruby 3.4.5

```bash
# инициализация

bin/rails db:migrate
bin/rails db:seed
```


```bash
# для запуска два разных окна

bin/rails s
yarn build:css --watch
```
